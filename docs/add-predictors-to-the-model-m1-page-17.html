4 Add Predictors to the Model: “M1” (page 17)

Level 1 Model (pupil \(i\))
\[POP_{ij} = B_{0} + B_{1} SEX_{ij} + B_{2} EXT_{ij} + e_{ij}\]

Level 2 Model (class \(j\))
no cross level interactions

Multilevel Model
\[ POP_{ij} = \gamma_{00} + \gamma_{10} SEX_{ij} + \gamma_{20} EXT_{ij} + \gamma_{02} YRS_j u_{0j} + u_{1j} SEX_{ij} + u_{2j} EXT_{ij} + e_{ij} \label{eq:M1} \tag{M1} \]
